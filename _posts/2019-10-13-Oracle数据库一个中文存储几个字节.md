# 前言

> **本文**主要记载了一个很容易忽视的知识点

# 前提介绍
在Oracle数据库里面，利用varchar2指定类型，那么一个中文占用几个字节？

# 答案
其实，这个跟数据库的编码有关，下面方法可以查询编码，并且查询中文占用几个字节。

1. 查看数据库编码

		select userenv('language') from dual;

2. 查看中文占用几个字节，假设表名为u，字段name为varchar2类型

		select u.name, length(u.name), lengthb(u.name) from u ;
